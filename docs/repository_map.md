# Repository Map

This repository is the companion code and source catalogue for the associated book on inverse problems and optimal transport.

## Top-level files

- `README.md`: repository overview, chapter map, direct source links, QR-code instructions, and citation guidance.
- `LICENSE`: license for self-written repository materials.
- `CITATION.cff`: citation metadata for the companion repository.
- `requirements.txt`: minimal pip-style dependency list.
- `environment.yml`: conda-style environment file.
- `check_report.txt`: repository consistency and verification report.

## Directories

- `chapters/`: chapter-organized teaching scripts and source notes.
- `docs/`: repository documentation and visual assets.
- `figures/qrcode/`: print-oriented QR-code output for the book manuscript.
- `third_party/`: external source catalogue, manuscript-synchronized BibTeX entries, and license notes.
- `tools/`: repository utility scripts.

## Source and attribution policy

Self-written teaching scripts are included directly in the relevant chapter folders. External public repositories are cited and linked for attribution and reproducibility, but their source code, datasets, model weights, trained models, and documentation are not redistributed here.
