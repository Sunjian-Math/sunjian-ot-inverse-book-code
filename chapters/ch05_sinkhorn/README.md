# Chapter 5 — Sinkhorn Benchmark

## Overview

This directory corresponds to the Chapter 5 practice section on differentiable Sinkhorn computation and entropic optimal transport.

## Contents

`main.py` is a self-written PyTorch script for standard Sinkhorn computation, epsilon scaling, and debiased Sinkhorn divergence.
