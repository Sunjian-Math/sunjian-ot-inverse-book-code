# Chapter 4 — Wasserstein Shift Landscape

## Overview

This directory corresponds to the Chapter 4 practice section comparing $L^2$, $W_1$, and $W_2$ behavior under a one-dimensional spatial shift.

## Contents

`main.py` is a self-written teaching script that computes normalized misfit landscapes for shifted Gaussian densities.
