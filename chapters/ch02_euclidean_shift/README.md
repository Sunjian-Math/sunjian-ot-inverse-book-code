# Chapter 2 — Euclidean Misfit under Spatial Shift

## Overview

This directory corresponds to the Chapter 2 practice section on pointwise Euclidean residuals under spatial displacement.

## Contents

`main.py` constructs a reference Gaussian signal and a shifted signal, computes pointwise residuals and an $L^2$ shift landscape, and visualizes the residual structure.
