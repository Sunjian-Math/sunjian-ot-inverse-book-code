# Chapter 3 — Discrete Optimal Transport Matrix Example

## Overview

This directory corresponds to the Chapter 3 practice section on a finite-dimensional optimal transport plan.

## Contents

`main.py` constructs two discrete distributions, builds a squared Euclidean cost matrix, and solves for the transport plan using POT.
